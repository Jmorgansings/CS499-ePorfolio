# CS499 ePortfolio for Jasmine Morgan

Welcome to my ePortfolio for the CS499 Capstone course. This repository showcases my mastery of key computer science concepts through the artifacts developed in my capstone project.

My capstone project, **Inv-Alert**, is an Android-based inventory management application designed to help users track products, quantities, and other details efficiently.

- **Link to the full project repository:** [Inv-Alert-Capstone](https://github.com/Jmorgansings/Inv-Alert-Capstone)

## Artifact Narratives

Below are the links to the detailed narratives for each of the core competency areas, as demonstrated in my capstone project.

1.  **[Software Design and Engineering](./software-design.md)**
    - This narrative explains the architectural design of the application, focusing on principles like Separation of Concerns and secure coding practices. The primary artifact is the `DataDisplayActivity.kt` class, which serves as the central UI controller.

2.  **[Algorithms and Data Structures](./algorithms-data-structures.md)**
    - This narrative discusses the use of data structures (`ArrayList`) and algorithms (database-driven sorting) to manage and display data efficiently. It highlights the choice to delegate sorting to the database for optimal performance.

3.  **[Databases](./databases.md)**
    - This narrative details the design and implementation of the application's SQLite database. The `DatabaseHelper.java` artifact demonstrates my ability to manage schema, handle CRUD operations, and write efficient, secure queries.
